# Ai Resume Analyzer
Hi, I am Akshit 
AI Resume Analyzer using Python & streamlit
Features
- Upload resume in PDF format
- Extract text from resume automatically
- Detect technical skills from resume and job description
- Calculate skill-based match percentage
- Highlight matched and missing skills

  Tech Stack
- Python
- Streamlit
- PyPDF2
- Scikit-learn

  How It Works
1. Upload a resume (PDF)
2. Paste a job description
3. The app extracts skills from both
4. A match score is calculated based on skill overlap
5. Results are shown with visual indicators
